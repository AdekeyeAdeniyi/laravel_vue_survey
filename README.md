# Survey Questionnaire Application

## Overview

This application allows users to create, manage, and respond to surveys. It is built using Laravel and Vue js.

## Features

-   Create and manage surveys
-   Respond to surveys
-   View survey results
-   User authentication

## Installation

### Prerequisites

-   Node.js and npm
-   Composer
